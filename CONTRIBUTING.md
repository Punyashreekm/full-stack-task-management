# Contributing to Collaborative Task Manager

Thank you for your interest in contributing! We want to keep the process simple and effective.

## Getting Started

1. **Fork the Repository**: Create your own copy of the project.
2. **Clone the Repo**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Collaborative-Task-Manager.git
   ```
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. **Create a Branch**:
   ```bash
   git checkout -b feature/your-feature-name
   ```

## Development Guidelines

- **Code Style**: Please follow consistent indentation and naming conventions.
- **Commits**: Write clear, descriptive commit messages.
- **Testing**: Ensure your changes don't break existing functionality. Run `npm run dev` to test locally.

## Submission Process

1. **Push Changes**:
   ```bash
   git push origin feature/your-feature-name
   ```
2. **Open a Pull Request**: Provide a clear description of your changes and why they are needed.
3. **Review**: Wait for review and address any feedback.

## Questions?

If you have any questions, feel free to open an issue or contact the maintainers.
